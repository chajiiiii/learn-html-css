CSS 선택자

가상 요소 선택자

가상 요소란?

실제 DOM에는 존재하지 않지만, CSS를 통해 가상으로 생성된 요소를 선택하여 스타일을 적용할 수 있습니다.

전체 선택자

전체 선택자는 모든 요소를 선택하여 스타일을 적용할 수 있습니다.

요소(tag) 선택자

요소 선택자는 HTML 요소 이름 즉 태그명을 사용하여 선택하고 스타일을 적용할 수 있습니다.

클래스 선택자

클래스 선택자는 요소의 클래스 명을 선택하여 스타일을 적용할 수 있습니다.

아이디 선택자

아이디 선택자는 요소의 아이디 명을 선택하여 스타일을 적용할 수 있습니다.

속성 선택자

속성 선택자는 요소의 속성의 유무나 속성 값의 조건에 따라 스타일을 적용할 수 있습니다.

자손 선택자

두 개 또는 그 이상의 선택자를 조합하여, 가장 마지막 선택자의 조상이 존재할 경우 해당 선택자에 스타일을 적용할 수 있습니다.
